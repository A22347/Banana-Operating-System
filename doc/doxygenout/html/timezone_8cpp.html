uint64_t SysTimezone ( regs *  r)

Gets or sets the timezone.

Parameters
ebxWhen setting the timezone, this is the timezone ID. This is the zero-based line number in the timezones file.
ecxSet to zero if getting the time zone, or any other value to set.
Returns
If setting the timezone, returns zero on success, or non-zero on failure. If getting the timezone, it returns the timezone ID, or -1 if this cannot be found.
