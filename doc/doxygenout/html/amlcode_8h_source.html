406/*
407 * Some of the flags and types below are of the form:
408 *
409 * AML_FLAGS_EXEC_#A_#T,#R, or
410 * AML_TYPE_EXEC_#A_#T,#R where:
411 *
412 * #A is the number of required arguments
413 * #T is the number of target operands
414 * #R indicates whether there is a return value
415 *
416 * These types are used for the top-level dispatch of the AML
417 * opcode. They group similar operators that can share common
418 * front-end code before dispatch to the final code that implements
419 * the operator.
420 */
421
