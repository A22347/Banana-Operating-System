14/*
15
16This module implements an ISO9660 file system for reading from a CDR or CD
17in the DC's GD-Rom drive.
18
19Rock Ridge support has now been implemented, thanks to Andrew Kieschnick
20who donated the code. Thanks to Bero for the Joliet support here.
21
22This FS is considerably simplified from what you'd find in a bigger kernel
23like Linux or BSD, since we have the pleasure of working with only a single
24device capable of ISO9660 at once =). So there are a number of things in here
25that are global variables that might otherwise not be.
26
27Some thanks are in order here to Marcus Comstedt for providing an ISO9660
28implementation that was easy enough to understand without downloading the
29full spec =). Thanks also in order to the creators of the BSD and Linux
30ISO9660 systems, as these were used as references as well.
31
32*/
